# Информационная система для салона красоты

Веб-приложение разработано в рамках дипломного проекта на тему **«Разработка информационной системы для салона красоты с функцией онлайн-бронирования услуг»**.

Система предназначена для автоматизации записи клиентов, управления услугами, мастерами и расписанием салона красоты.

## Основные возможности

### Для клиента

* просмотр услуг и информации о мастерах;
* онлайн-запись на выбранную услугу;
* выбор даты и свободного временного слота;
* расчёт свободного времени с учётом продолжительности процедуры;
* защита от пересечения записей клиента;
* просмотр и отмена собственных записей;
* редактирование данных профиля;
* просмотр уведомлений;
* общение с администратором через чат;
* оформление подарочных сертификатов.

### Для администратора

* управление услугами и мастерами;
* просмотр и изменение статусов записей;
* управление расписанием;
* перенос записей клиентов;
* работа с сообщениями и уведомлениями;
* управление сертификатами;
* просмотр статистики и финансовой информации;
* редактирование данных сайта.

## Используемые технологии

* HTML5;
* CSS3;
* JavaScript;
* PHP;
* MySQL;
* Open Server.

## Установка и запуск

1. Установить и запустить Open Server.
2. Поместить папку проекта в каталог доменов Open Server.
3. Создать базу данных MySQL.
4. Импортировать SQL-файл базы данных из проекта.
5. Проверить параметры подключения к базе данных в конфигурационном PHP-файле.
6. Перезапустить Open Server.
7. Открыть сайт через локальный адрес проекта в браузере.

## Назначение проекта

Программа позволяет сократить нагрузку на администратора, избежать двойного бронирования времени, упростить взаимодействие между клиентами и сотрудниками салона, а также обеспечить удобную онлайн-запись в любое время суток.
